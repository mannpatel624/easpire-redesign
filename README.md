# 🚀 Easpire Website  
### React + Vite + Convex

This repository contains the official frontend for the **Easpire Website**, built with modern tools including **React**, **Vite**, **pnpm**, and **Convex** for backend services.

---

## 🧰 Tech Stack

- **React** – Frontend library
- **Vite** – Fast build & dev server
- **pnpm** – Package manager
- **Convex** – Backend & database
- **Node.js** – Runtime environpnpm installment

---

## 📦 Prerequisites

Before running the project, make sure you have the following installed:

- **Node.js** `v20.19+` or `v22.12+`  
- **pnpm** (package manager)
- **Git**
- **Convex account** (for backend services)

### 🔍 Check Installed Versions

```bash
node -v
pnpm -v
npx convex dev
npx pnpm dev
